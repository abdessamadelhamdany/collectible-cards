## To run this project

Please check this docs page to setup [Laravel Sail](https://laravel.com/docs/8.x/sail)

### After aliasing sail:

`sail up`

`sail composer install`

`sail artisan migrate --seed`

`sail npm install`

`sail npm run dev`
